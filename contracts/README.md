# FlowGuard Contracts

Layla CHIPs smart contracts for FlowGuard treasury management.

## Structure

- `FlowGuard.cash` - Main covenant
- `loops.cash` - Loop module for recurring unlocks
- `bitwise.cash` - Bitwise state encoding
- `functions.cash` - Reusable function modules
- `tests/` - Test files

## Development

```bash
# Compile contracts
pnpm build

# Run tests
pnpm test

# Check syntax
pnpm check
```
